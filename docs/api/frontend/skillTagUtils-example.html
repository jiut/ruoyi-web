技能标签工具类使用示例

基础使用 - 获取标签分类和样式

展示不同分类的技能标签：

// 后端返回英文简写格式，前端自动转换显示为中文 const tags = ['figma', 'interaction_design', 'user_experience']; SkillTagUtils.renderTags('#basic-demo', tags);

默认主题展示

统一的默认主题样式：

// 后端返回英文简写，前端自动转中文并按分类着色 const tags = ['sketch', 'ui_design', 'prototype_design', 'photoshop']; SkillTagUtils.renderTags('#theme-demo', tags);

分组显示

按照分类分组显示技能标签：

// 不同分类的英文简写标签，自动分组显示 const mixedTags = ['figma', 'sketch', 'interaction_design', 'ui_design', 'user_experience', 'prototype_design']; SkillTagUtils.renderGroupedTags('#grouped-demo', mixedTags);

自定义样式

使用自定义样式选项：

const customOptions = { baseClasses: 'px-3 py-1.5 rounded-full text-sm font-medium', gap: 'gap-3' }; SkillTagUtils.renderTags('#custom-demo', tags, 'default', customOptions);

实时分类检测

// 输入英文简写，检测分类并获取中文显示名 const category = SkillTagUtils.getTagCategory('figma'); const displayName = SkillTagUtils.getTagDisplayName('figma'); console.log(category); // 输出: 'tool' console.log(displayName); // 输出: 'Figma'

统计信息

获取技能标签的分类统计：

const stats = SkillTagUtils.getCategoryStats(tags); // 返回: { tool: 2, field: 1, skill: 1 }

Vue.js 组件示例

<template> <div class="skill-tags"> <span v-for="tag in skills" :key="tag" :class="getTagClasses(tag)" > {{ tag }} </span> </div> </template> <script> export default { props: ['skills'], methods: { getTagClasses(tag) { return SkillTagUtils.getTagClasses(tag, 'default'); } } } </script>
